## 🇷🇺 WC Image Mirror — зеркалирование изображений WooCommerce

**WC Image Mirror** — это удобный плагин для WordPress / WooCommerce, который позволяет **массово зеркалировать изображения товаров по горизонтали прямо из админки**, без изменения имен файлов и структуры медиа-библиотеки.

Идеально подходит для интернет-магазинов, где требуется быстро изменить визуал товаров (например, обувь, одежда, техника и т.д.).

---

### 🚀 Основные возможности

#### 🪞 Массовое зеркалирование изображений
- Горизонтальное отражение изображений товаров
- Обработка сразу нескольких товаров
- Быстрая работа без перезагрузки страницы

#### 📂 Без изменения файлов
- Имена файлов остаются прежними
- Не создаются дубликаты
- Работа напрямую с существующими изображениями

#### ⚡ Интеграция в админку WooCommerce
- Удобный доступ из панели администратора
- Простое управление без сложных настроек
- Нативная интеграция в WordPress

---

### ⚙️ Как это работает

- Плагин использует встроенные возможности WordPress для обработки изображений
- Изображение изменяется (зеркалится) без изменения URL
- Все изменения происходят безопасно внутри системы

### 📦 Установка

1. Скачайте или клонируйте репозиторий
2. Загрузите папку `wc-image-mirror` в:

/wp-content/plugins/

3. Активируйте плагин в админке WordPress
4. Перейдите в WooCommerce и используйте функционал зеркалирования

## 🇺🇸 WC Image Mirror — WooCommerce Image Mirroring Plugin

**WC Image Mirror** is a lightweight WordPress plugin that allows you to **bulk flip WooCommerce product images horizontally directly from the admin panel**, without renaming files or duplicating media.

Perfect for e-commerce stores that need fast visual adjustments.

---

### 🚀 Features

#### 🪞 Bulk Image Mirroring
- Horizontally flip product images
- Process multiple products at once
- Fast and efficient

#### 📂 No File Renaming
- Keeps original filenames
- No duplicates created
- Works with existing media library

#### ⚡ WooCommerce Integration
- Built directly into admin panel
- Easy to use
- No complex setup required

---

### ⚙️ How It Works

- Uses native WordPress image processing
- Modifies images without changing URLs
- Safe and efficient

---

### 🧠 Benefits

- Saves time on bulk edits  
- Clean and simple workflow  
- No extra files  
- Optimized for WooCommerce  

---

### 🛠️ Suitable for

- WooCommerce stores  
- Clothing & footwear shops  
- Product catalogs  
- Visual-heavy websites  

---

### 📦 Installation

1. Download or clone repository  
2. Upload to:

/wp-content/plugins/

3. Activate plugin  
4. Use from WooCommerce admin panel
